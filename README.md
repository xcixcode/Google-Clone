# Google-Clone

**Google 메인 페이지의 사용자 인터페이스를 HTML, CSS, JavaScript만을 사용하여 클론한 웹 개발 연습 프로젝트**  

HTML, CSS, JavaScript만을 사용하여 Google 메인 페이지의 UI를 재현한 프로젝트입니다.  
jQuery를 활용해 동적인 UI 동작을 구현했으며, 다크/라이트 모드 전환, 검색어 입력 처리, 버튼 클릭 이벤트 등 실사용 기능을 일부 반영했습니다.

---

## 개요

**Google-Clone** 프로젝트는 Google 메인 페이지 UI를 클론하여 HTML, CSS, JavaScript 활용 능력을 향상시키기 위해 제작되었습니다.  
사용자가 검색어를 입력하거나 버튼을 클릭하면, 실제 Google 검색 페이지와 연동되는 동작을 일부 구현했습니다.  
다크/라이트 모드 전환, 실시간 입력 감지, UI 버튼 이벤트 등 인터랙티브 기능 학습에 초점을 맞춘 프로젝트입니다.

---

## 기술 스택

- **HTML5** - 페이지 구조 구성 및 콘텐츠 배치  
- **CSS3** - 레이아웃, 반응형 디자인, 버튼/아이콘 스타일링  
- **JavaScript (Vanilla + jQuery)** - 이벤트 처리 및 UI 동작 제어  
- **FontAwesome** - 키보드, 마이크, 이미지 검색 아이콘 적용

---

## 주요 기능

- Google 메인 페이지 레이아웃 클론  
- 검색 입력창 및 버튼 인터랙션 구현  
- 키보드/마이크/이미지 검색 아이콘 포함  
- "Google 검색" 및 "I'm Feeling Lucky" 버튼 구현  
- 실시간 검색어 입력 감지 및 clear 버튼 표시  
- Enter 키 입력 시 실제 Google 검색 페이지로 이동  
- 다크/라이트 모드 전환 (버튼 클릭으로 색상 및 로고 전환)

## 설치 및 실행

### GitHub Pages
https://xcixcode.github.io/Google-Clone/

### 로컬에서 실행
1. 저장소를 클론하거나 `.zip`으로 다운로드합니다.  
2. 로컬에서 `index.html` 파일을 웹 브라우저로 열면 바로 실행됩니다.  
3. 인터넷 연결 필요 (jQuery, FontAwesome CDN 사용)
